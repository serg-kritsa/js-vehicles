Задание:
Существует три вида транспортных средств (ТС): автомобиль, самолет и катер. 
У них у всех есть как общие параметры (максимальная скорость, вместимость), 
так и индивидуальные (например, тип кузова у автомобиля, размах крыла у самолета и максимальная мощность мотора для катера).

На сервере хранится файл в формате JSON, в котором содержится информация о ТС (см. vehicles.json).

Программа должна 
загрузить с сервера список ТС, 
создать и заполнить соответствующие коллекции объектов, 
а затем отобразит данные из коллекций в соответствующих таблицах на странице.

При разработке использовать приложенную HTML страницу (см. index.html).

Дополнительные задачи:
* Валидация данных
* Обработка ошибок
* Unit-тесты, покрывающие некоторый функционал

Дополнительные требования и ограничения:
* Использовать чистый JavaScript стандарта ES5 без сторонних библиотек и фреймворков (включая jQuery)
* В задаче подразумевается использование наследования
* Поддержка браузеров: IE9+, FF4+, Chrome 20+, Safari 5+
